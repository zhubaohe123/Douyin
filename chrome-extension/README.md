# 🔌 抖音评论定位助手 - 谷歌浏览器插件使用指南

这是一个为您度身定制的 Chrome 浏览器开发者插件。配合您的**抖音评论采集分析系统**使用，能实现在抖音作品页面**“自动平滑滚动定位”**并**“青蓝色霓虹呼吸框高亮”**指定的评论或回复！

---

## 🛠️ 第一步：安装插件至谷歌浏览器 (Chrome)

由于这是为您定制的专用扩展，您无需去 Chrome 应用商店，可以直接以**开发者模式**加载：

1. 打开 **Google Chrome 浏览器**。
2. 在地址栏输入 `chrome://extensions/` 并回车（或点击浏览器右上角 “三点” -> “更多工具” -> “扩展程序”）。
3. 在扩展程序页面右上角，将 **“开发者模式” (Developer mode)** 开关**开启**。
4. 点击页面左上角的 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
5. 在弹出的文件夹选择框中，选择您电脑上的该插件目录：
   👉 **`D:\daima\Douyin\chrome-extension`**
6. 点击确定，您将在列表中看到 **“抖音评论定位助手”** 插件已成功启用！

---

## 🔌 第二步：验证连接状态

1. 启动您的本地 React 开发服务器（如 `npm run dev`）并打开采集系统网页（默认地址通常是 `http://localhost:5173`）。
2. **观察系统提示**：
   - 网页加载完成后，系统会自动完成插件握手。
   - 网页右上角将滑出绿色成功提示：**“已成功对接抖音评论采集助手插件 🔌”**。
   - 所有评论的跳转定位按钮将瞬间从**粉红色渐变**（降级跳转模式）蜕变为高对比度的**青蓝色霓虹配色**（插件联动模式）！
   - 主评论卡片按钮显示为 `🔌 自动精准定位`，二级子回复卡片按钮显示为 `🎯 精准定位`。

---

## 🎯 第三步：使用精准定位

1. 在采集系统中抓取视频评论，并在列表中找到您感兴趣的某条评论（或其底部的二级回复）。
2. 点击对应的 **`🔌 自动精准定位`**（主评论）或 **`🎯 精准定位`**（子回复）按钮。
3. **系统将自动执行以下操作**：
   - 将该评论的元数据（作品ID、评论ID、作者昵称、评论摘要文本、父评论ID）安全打包并发送给插件后台。
   - 系统发出提示：*“正在通过浏览器插件定位至该评论...”*。
   - 浏览器自动在**新标签页**中打开该抖音视频播放页 (`douyin.com/video/...`)。
4. **抖音页面的神奇效果**：
   - 页面加载后，插件将自动检测到该视频的挂起定位请求。
   - 插件将接管滚动条，**自动平滑向下滚动**（若有评论抽屉，会自动滚动评论抽屉并触发懒加载），直至目标评论刚好呈现在屏幕正中央。
   - 目标评论卡片将被套上一个**酷炫的青蓝色霓虹呼吸灯高亮边框**，并伴随柔和的淡蓝色底色，持续展示 5 秒，让您的视线在一瞬间锁定目标！

## 💡 常见问题排查

- **Q: 为什么按钮还是粉红色的？没有提示连接成功？**
  - **A**: 请检查 `chrome://extensions/` 页面里该插件是否被关闭了，或者点击插件卡片上的“刷新”图标重新加载一下。然后刷新您的采集系统网页即可。
- **Q: 为什么跳转到抖音后没有自动滚动？**
  - **A**: 抖音网页版加载速度受网络影响较大。插件默认会持续检索定位目标 15 秒（每 0.5 秒检测一次）。如果网络极慢，您可以尝试手动在抖音页面向下滚动一下（触发抖音拉取接口），插件一旦检测到该评论便会立即执行高亮定位。
